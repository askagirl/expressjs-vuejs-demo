# expressjs-vuejs-demo
demo a web page use vue.js and express.js

# Run
```
node server.js
```

## Show Vue.js Demo
```
http://localhost:3000/
```

## Show apidoc Demo
```
http://localhost:3000/doc
```

# Generate API Doc
```
// install apidoc
npm install -g apidoc
```

```
// output apidoc
apidoc -i api -o assets/doc
```
