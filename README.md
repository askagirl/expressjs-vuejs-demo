# expressjs-vuejs-demo
demo a web page use vue.js and express.js

# Run
```
node server.js
```

# Generate api doc
```
// install apidoc
npm install -g apidoc
```

```
// output apidoc
apidoc -i api -o assets/doc
```

## Show vue.js demo
```
http://localhost:3000/
```

## Show api demo
```
http://localhost:3000/api/v1/user
```

## Show apidoc demo
```
http://localhost:3000/doc
```
