# expressjs-vuejs-demo
demo a web page use vue.js and express.js

## Required
- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.org/)

## Run
```
node server.js
```

## Demo
```
http://localhost:3000/
```

## Generate API Document
```
// install apidoc
npm install -g apidoc
```

```
// output apidoc
apidoc -i api -o doc
```

## View API Document
```
http://localhost:3000/doc
```
